**License:** [GPLv3][license]

Tools for 1D and 2D genomic significance analysis

Installation
------------
```
reuse -q GCC-4.9 ### if on Broad Institute servers, add GCC-4.9
git clone --recursive https://github.com/jwalabroad/ginseng
cd gingeng
./configure
make
```
